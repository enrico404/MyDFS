Modalità di sviluppo

Per ogni nodo
	1) crea una cartella nella tua home chiamata "shDir"

Attenzione: il nome dell'utente deve essere lo stesso per tutti quanti i nodi



Istruzioni per utilizzo/compilazione:

Server:
	- apri un terminale e spostati nella cartella src
	- avvia rmiregistry
	- apri un altro terminale e spostati in out
	- esegui:

	java -Djava.security.policy=./perm.policy Server/ServerClass <dir_condivisa>

serverManager:
	- avvia rmregistry se non è già stato avviato
	- apri un terminale in out ed esegui:

	java -Djava.security.policy=./perm.policy Server/ServerManager //<serverX_ip>/<serverX_name>



Client: 
	- apri un terminale e spostati nella cartella out
	- esegui:

	java Client.ClientClass //<serverManager_ip>/ServerManager




Per compilare: 
	- apri un terminale in src
	- esegui:

	javac -d ../out/ Server/ServerInterface.java Server/ServerClass.java Client/ClientClass.java Server/ServerManagerInterface.java Server/ServerManager.java utils/utils.java utils/MyFileType.java utils/ConsoleColors.java utils/Converter.java Client/ParamParser.java
 

ATTENZIONE:
    il file perm.policy deve trovarsi una cartella indietro dei file binari (c'è una copia di questo file nella carrtella src)
