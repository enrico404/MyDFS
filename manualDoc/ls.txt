Resistituisce i file nella directory corrente

Uso:
	ls [-l, -d, -h]..

Opzioni:
	-l -> per vedere anche i meta dati dei file
	-d -> per calcolare anche la dimensione delle directory
	-h -> per rendere l'output delle dimensioni human readable

Esempi d'uso:
    ls -l
    ls -d
    ls -l -d
    ls -l -d -h
    ls -ldh
    ls -lh
